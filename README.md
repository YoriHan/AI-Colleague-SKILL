`#gtm_friends` 频道里各 AI 同事的 skill 汇总。每个 skill 文件记录该同事的角色、能力、自动化任务与首次配置引导。

| AI 同事 | 角色（一句话） | 一键安装 |
|---|---|---|

| **SEO Genius** (@seogenius-2) | SEO/GEO：关键词策略、页面 SEO、内容缺口、自然增长、SEO 发布把关 | ✅ [`/seo-genius`](./seo-genius/SKILL.md) |
| **Day Planner** (@dayplanner-2) | 日程规划：把优先级/会议/跟进变成能过的一天，保护注意力、让时间可见 | ✅ [`/day-planner`](./day-planner/SKILL.md) |
| **Email Genius** (@emailgenius-2) | 邮件自动化：巡检 Gmail、分诊、起草英文回复、批量触达、发票跟踪、导入 Notion | ✅ [`/email-genius`](./email-genius/SKILL.md) |

## 安装

Helio 安装 skill：`heliox skill install <链接>`。仓库里名为 `SKILL.md`（带 `name` + `description` frontmatter）的即可一键安装。

```
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/gatlin
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/seo-genius
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/day-planner
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/email-genius
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/userguardian
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/official-account-assistant
heliox skill install https://github.com/YoriHan/AI-Colleague-SKILL/tree/main/trace
```
